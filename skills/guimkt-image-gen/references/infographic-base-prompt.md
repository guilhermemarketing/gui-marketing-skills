# Prompt-Base — Infográficos Punk Zine

Este é o prompt-base para **infográficos** (dados, comparativos, fluxos, diagramas).
Diferente das ilustrações editoriais, infográficos **contêm texto/dados** como parte
da composição. O estilo visual permanece neo-brutalist punk.

## Prompt-Base para Infográficos (copiar integralmente)

```
Mixed-media editorial infographic with punk zine energy and neo-brutalist graphic accents. Data visualization layout with raw, hand-made aesthetic — like a punk zine protest poster, not polished corporate design.

STYLE: Thick solid black outlines (4-6px strokes) and hard solid black offset drop shadows (X+Y offset, zero blur, 100% opacity). Rough geometric shapes, bold arrows, crude circles, tape strips, staple marks. Paper grain and xerox noise texture on background. Halftone dot patterns as accents. Everything feels hand-ripped, analog, confrontational.

COLORS: Flat solid fills only. Background: Creamy Off-white (#f7f3ed) with subtle paper grain. Accents: Electric Purple (#864df9) and Pastel Yellow (#F7E397). Black (#000000) for outlines, text, and shadows. No gradients on backgrounds. Ink splatter marks and tape strips as decorative analog accents.

MOOD: Raw, provocative, intellectually aggressive — Bloomberg Businessweek confrontational cover meets punk zine. Professional enough for B2B marketing, rebellious enough to stop the scroll.

FORBIDDEN: No 3D rendering, no glass/glossy effects, no realistic lighting. No childish or cartoonish style. No corporate clip art. No clean polished vector illustration. No cute or friendly aesthetic.
```

## Diferenças vs Ilustrações

| Aspecto | Ilustração | Infográfico |
|---------|-----------|-------------|
| Texto na imagem | ❌ Proibido | ✅ Esperado (dados, labels, títulos) |
| Idioma do texto | — | Português (conteúdo) + Inglês (prompt) |
| Espaço limpo | 30-40% obrigatório | Conforme necessário pelo layout |
| Elemento dominante | Foto cutout | Diagrama/dados |
| Decoração punk | Secundária | Integrada (dividers, acentos entre dados) |

## Tipos de Infográfico Suportados

### Split-Screen Comparativo

Dois lados divididos por lightning bolt, torn-paper line, ou crude divider.
Ideal para: vs comparisons, antes/depois, dados antagônicos.

```
LAYOUT: Split-screen comparison. LEFT side: [dados positivos com setas up].
RIGHT side: [dados negativos com setas down]. A bold crude hand-drawn
lightning bolt or rough torn line divides the two sides. Title at top.
Bottom quote or conclusion.
```

### Flow Vertical (Cadeia de Valor)

Estágios conectados por setas grossas, separados por torn-paper dividers.
Ideal para: processos, funis, etapas sequenciais, causa→efeito.

```
LAYOUT: Title at top. [N] connected stages flowing vertically, each with a
crude hand-drawn icon and bold arrow connecting to the next stage.
Each stage separated by rough torn-paper dividers.
```

### Venn Diagram

Círculos desenhados com brushstrokes grossos, interseção com crosshatch.
Ideal para: conceitos complementares, fusão de áreas, sobreposição.

```
LAYOUT: Title at top. A large hand-drawn Venn diagram with two overlapping
crude circles drawn with thick rough brushstrokes. LEFT circle labeled
"[CONCEITO A]" in Electric Purple. RIGHT circle labeled "[CONCEITO B]"
in Pastel Yellow. OVERLAP zone has crude crosshatch pattern and is labeled
"[RESULTADO]". Quote below the diagram.
```

### Data Highlight

Um dado central em destaque massivo, contexto ao redor.
Ideal para: estatísticas impactantes, KPIs, números de destaque.

```
LAYOUT: One massive number or data point centered, with thick black outlines
and offset drop shadow. Supporting context data smaller around it. Crude
hand-drawn accent shapes (circles, arrows, starbursts) highlighting the
main number.
```

## Paleta (mesma da ilustração)

| Cor | Hex | Uso em infográficos |
|-----|-----|---------------------|
| Creamy Off-white | `#f7f3ed` | Fundo principal |
| Pure Black | `#000000` | Outlines, texto, sombras |
| Electric Purple | `#864df9` | Destaques primários, ícones, círculos |
| Pastel Yellow | `#F7E397` | Highlights de dados, acentos |
| Mint Green | `#b0e4da` | Setas positivas, acentos pontuais |
| Light Lilac | `#dcd3ef` | Fundos alternativos |
| Soft Pink | `#efcde5` | Acentos pontuais |
